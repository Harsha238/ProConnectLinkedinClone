🚀 ProConnect – LinkedIn Clone

A full-stack social networking platform inspired by LinkedIn, built using modern web technologies.
Users can connect, share posts, and interact in a clean and responsive interface.

---

🌐 Live Demo

🔗 Frontend: https://pro-connect-linkedin-clone-seven.vercel.app
🔗 Backend: https://proconnectlinkedinclone-2.onrender.com

---

✨ Features

- 🔐 User Authentication (Register / Login)
- 👤 User Profiles
- 📝 Create & View Posts
- 🤝 Connect with other users
- 📸 Image Upload Support
- ⚡ Real-time API Integration
- 📱 Fully Responsive Design

---

🛠️ Tech Stack

Frontend

- ⚛️ Next.js
- 🎨 CSS Modules
- 🔗 Axios

Backend

- 🟢 Node.js
- 🚀 Express.js
- 🍃 MongoDB (Atlas)
- 🧩 Mongoose

Deployment

- 🌍 Vercel (Frontend)
- ☁️ Render (Backend)
- 🗄️ MongoDB Atlas (Database)

⚙️ Environment Variables

Create a ".env" file in backend:

PORT=9080
MONGO_URI=your_mongodb_connection_string

---

▶️ Run Locally

1. Clone the repo

git clone https://github.com/Harsha238/ProConnectLinkedinClone.git
cd ProConnectLinkedinClone

2. Backend setup

cd backend
npm install
npm run dev

3. Frontend setup

cd frontend
npm install
npm run dev
